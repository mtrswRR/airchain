# AirChain

AirChain — платформа мониторинга качества воздуха с валидацией данных в блокчейн.

## Цель проекта
Создание прототипа системы, обеспечивающей:
- сбор данных о качестве воздуха от распределённых сенсоров;
- валидацию и проверку целостности данных;
- фиксацию хэшей данных в блокчейн;
- предоставление пользователям достоверной информации о состоянии воздуха.

## Вертикальный прототип (scope)

В рамках вертикального прототипа реализуется **один сквозной сценарий**:

1. Приём данных от сенсора (PM2.5, PM10, CO₂).
2. Базовая валидация данных.
3. Формирование хэша набора данных.
4. Запись хэша в блокчейн (эмуляция / mock).
5. Просмотр статуса валидации данных.

## Функции прототипа
- REST API для приёма данных от сенсоров;
- сервис валидации данных;
- модуль блокчейн-валидации (упрощённая модель);
- API для получения статуса валидации.

## Вне рамок прототипа
- реальная блокчейн-нода;
- мобильное приложение;
- система токенов;
- масштабирование и отказоустойчивость.

## Технологии (план)
- Python
- FastAPI
- Pydantic
- SQLite (или in-memory)
